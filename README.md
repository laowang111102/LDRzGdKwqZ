# 项目简介 python50

本项目是基于 Django 框架开发的超市在线销售与分析系统，主要使用 Python 语言，结合了 Django 框架、echarts 和 MySQL 数据库等技术。

## 技术栈

- Python
- Django
- echarts
- MySQL

## 功能模块

- **销售数据可视化分析**：通过图表形式直观展示销售数据。
- **商品分类**：对商品进行分类管理，便于用户浏览和搜索。
- **购物车**：用户可随时查看已选商品，并进行管理。
- **结算系统**：支持商品结算，生成订单。
- **订单查询**：用户可以查询自己的订单状态和历史记录。
- **个人中心**：用户可以管理个人信息，查看订单状态等。
- **商品详情**：展示商品的详细信息，便于用户了解和选择。

## 系统角色

系统主要包含以下角色：

- **普通用户**：浏览商品、购买商品、管理个人订单等。
- **管理员**：管理商品信息、处理订单、分析销售数据等。

## 运行环境

- Python 环境配置要求：3.6 以上版本
- 数据库：MySQL

## 部署步骤

1. 配置 Python 环境和必要的库。
2. 安装 Django 并初始化项目。
3. 创建 MySQL 数据库，并执行提供的 SQL 文件以初始化数据。
4. 配置项目的数据库连接和静态资源。
5. 运行项目。
